Git

Populiariausia kodo versijavimo programa.

Git komandos

  • git init - sukuria lokalią git'o repozitoriją dabartiniame aplankale.
  • add - prideda nurodytus failus prie commit'ui paruoštų failų (stage'ina).
  • commit -m "message" - sukuria kodo versiją iš stage'intų failų ir prideda privalomą žinutę apie pakitimus(žinutė turėtų būti ne ilgesnė nei 50 simbolių).
