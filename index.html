Git komandos

  • git init - sukuria lokalią git'o repozitoriją dabartiniame aplankale.
  • git add - prideda nurodytus failus prie commit'ui paruoštų failų (stage'ina).
  • git commit -m "message" - sukuria kodo versiją iš stage'intų failų ir prideda privalomą žinutę apie pakitimus(žinutė turėtų būti ne ilgesnė nei 50 simbolių).
  • git status - nurodo informaciją paie tavo lokalios git'o repozitorijos statusą.
  • git push - iš lokalios repozitorijos išsiunčia commit'us į išorinę repozitoriją.
  • git clone - klonuoja nurodytą repozitoriją iš išorinės repozitorijos.
  • git pull - iš išorinės repozitorijos atsiunčiami commit'ai į lokalią repozitoriją.
